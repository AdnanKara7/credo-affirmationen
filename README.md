# Credo – Affirmationen & Zitate 📱✨  

## ✨ Beschreibung  
Credo ist eine minimalistische iOS-App, die tägliche Affirmationen und inspirierende Zitate liefert. Sie unterstützt Nutzer dabei, eine positive Denkweise zu entwickeln und ihre Ziele mit mehr Fokus zu verfolgen.  

## 🚀 Features  
- 🎤 Zufällige Affirmationen & inspirierende Zitate  
- 🎨 Minimalistisches und benutzerfreundliches Design  
- 📢 Erinnerungsfunktion für tägliche Motivation  
- 🌙 Dark Mode Unterstützung
